  INPUT length
  INPUT width
  area = length * width
  OUTPUT area

### **2. Determine if a number is even or odd**

#### **Pseudocode:**
```
START
  INPUT number
  IF number MOD 2 == 0 THEN
    OUTPUT "Even"
  ELSE
    OUTPUT "Odd"
  END IF
END
```

### **3. Find the largest of three given numbers**

#### **Pseudocode:**
```
START
  INPUT num1, num2, num3
  IF num1 >= num2 AND num1 >= num3 THEN
    largest = num1
  ELSE IF num2 >= num1 AND num2 >= num3 THEN
    largest = num2
  ELSE
    largest = num3
  END IF
  OUTPUT largest
END
```
